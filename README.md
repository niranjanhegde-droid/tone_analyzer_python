# 🎙️ ToneScope – AI Meeting Tone Analyzer

ToneScope is an AI-powered meeting tone analysis web application that analyzes meeting conversations and identifies the overall emotional tone using Natural Language Processing (NLP). The application provides real-time insights through an interactive dashboard, helping users understand communication patterns and improve meeting effectiveness.

---
## 🌐 Live Demo

[view here](https://tonescope-meeting-tone-analyzer.onrender.com )

## 📌 Features

- 🎤 Upload meeting transcripts (.txt)
- 🤖 AI-powered tone analysis
- 😊 Detects Positive, Neutral, and Negative sentiments
- 📊 Interactive analytics dashboard
- 📈 Visual charts and statistics
- 📝 Sentence-wise sentiment analysis
- 📂 Meeting history storage using SQLite
- 🌙 Modern responsive user interface
- ⚡ Fast Flask backend

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/ToneScope-Meeting-Analyzer.git

cd ToneScope-Meeting-Analyzer
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
